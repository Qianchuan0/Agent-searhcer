# 网站

本网站使用 [Docusaurus 2](https://docusaurus.io/) 构建，这是一个现代静态网站生成器。

## 前置条件

要在本地构建和测试文档，请先下载安装 [Node.js](https://nodejs.org/en/download/)，然后安装 [Yarn](https://classic.yarnpkg.com/en/)。
在 Windows 上，你可以通过随 Node.js 附带的 npm 包管理器安装：

```console
npm install --global yarn
```

## 安装

```console
pip install pydoc-markdown
cd website
yarn install
```

## 本地开发

进入 website 文件夹并运行：

```console
pydoc-markdown
yarn start
```

该命令会启动本地开发服务器并打开浏览器窗口。大多数修改会实时生效，无需重启服务器。
